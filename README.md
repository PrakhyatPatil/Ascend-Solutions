# Team Ascend

We are **Team Ascend**, a group of student developers from Acropolis Institute of Technology and Research.

---

## Problem Statement

**Accessible Navigation for People with Disabilities**

Millions of people with visual or mobility impairments struggle to navigate public spaces safely and independently. Existing navigation tools are largely built for able-bodied users and lack the real-time awareness, audio-first guidance, and detailed accessibility information that disabled individuals need.

We are building a real-time accessible navigation system that uses audio cues, hazard detection, and crowd-sourced accessibility data to help people with visual or mobility impairments navigate the world with greater confidence and independence.
