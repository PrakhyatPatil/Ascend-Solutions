NAVABLE - HACKATHON BUILD CONTEXT (single source of truth)

DATE
- April 25, 2026

NORTH STAR
- Build a voice-first accessibility navigation app for India that helps visually and mobility-impaired users move independently and safely.

ONE-LINE PITCH
- Navable detects hazards in real time using on-device AI, answers navigation questions by voice, and uses trusted delivery-partner data to map accessibility across the city.

TARGET USERS
- Visually impaired users
- Mobility-impaired users
- Initial demo geography: Indore

MVP FOR JUDGES (must work in live demo)
1) Live hazard detection from camera with spoken alerts
2) Accessibility map with color-coded pins (pre-seeded data is acceptable)
3) Big mic conversational AI agent (Hindi + English acceptable)
4) SOS action that sends location to trusted contact

CAN BE MOCKED IF TIME IS LOW
- Delivery partner contribution form
- Full live family tracking map
- Polished multilingual tuning
- User-submitted pin workflow

CORE ARCHITECTURE
- Mobile app: React Native (fastest path for 24h)
- Hazard AI: YOLOv8-nano style model, on-device TFLite inference
- Voice pipeline: STT -> LLM -> TTS
	- STT: AWS Transcribe
	- LLM: Amazon Nova Lite on Bedrock
	- TTS: AWS Polly
- Backend: AWS Lambda + API Gateway
- Data: DynamoDB (pins + users)
- Maps: Google Maps SDK with accessibility overlay
- SOS: AWS SNS (or equivalent SMS mechanism)

ACCESSIBILITY PIN DATA MODEL (minimum fields)
- place_id
- place_name
- lat, lng
- entry_access: ramp | step | both | neither
- lift_status: working | broken | absent
- path_quality: smooth | uneven | under_construction
- blockage: clear | blocked
- accessible_toilet: yes | no
- score: 1-5
- last_updated_at

HAZARD EVENT PAYLOAD (minimum)
{
	"hazard": "vehicle",
	"confidence": 0.87,
	"distance_estimate": "near",
	"direction": "left",
	"alert_level": "high"
}

UI/UX PRIORITIES
- Large central mic button
- Large touch targets and high contrast
- Voice-first flow, minimal typing
- Clear map pin legend: green/amber/red/grey
- Fast and obvious SOS action

DEMO SCRIPT (90s)
